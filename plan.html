Mauritanian License Plate Recognition [cite: 17]

Goal: Build a custom YOLOv8 Model & Pre-processing Pipeline

Monday: The Surgeon's Knife (OpenCV) 2 Hours

Goal: Master image manipulation. You cannot train the AI if you cannot handle the image matrices.

  • Install libraries: pip install opencv-python ultralytics torch numpy
  • Script 1: Load a car image using cv2.imread.
  • Script 2: Convert to Grayscale (simplifies data for OCR later).
  • Script 3: Perform a manual "crop" using NumPy slicing (e.g., img[y1:y2, x1:x2]).
import cv2 img = cv2.imread('car.jpg') # Crop syntax: image[y_start:y_end, x_start:x_end] plate = img[300:400, 200:500] cv2.imwrite('cropped_plate.jpg', plate)

Wednesday: The Grind (Annotation I) 2 Hours

Goal: Create the Ground Truth for YOLO.

  • Set up account on Roboflow (recommended) or install LabelImg.
  • Upload your best 50-100 Mauritanian car images.
  • Labeling: Draw tight boxes around plates. Class name: license-plate.
Tip: Be consistent. Do not include the car bumper. Only the white/yellow plate area.

Saturday: Training & Debugging 5 Hours

Goal: Train the Phase 1 Model (Object Detection)[cite: 42].
